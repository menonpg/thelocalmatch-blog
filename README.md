# The Local Match Co. — Blog

Blog for [The Local Match Co.](https://thelocalmatch.com) — professional matchmaking franchise.

Built with [Astro](https://astro.build). Deployed to GitHub Pages at [blog.thelocalmatch.com](https://blog.thelocalmatch.com).

## Development

```bash
npm install
npm run dev
```

## Deploy

Push to `main` — GitHub Actions handles the rest.
